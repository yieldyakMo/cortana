# Agent Jobs

Create one directory per coordinated task:

`operations/agent-jobs/<YYYYMMDD>-<short-name>/`

Recommended contents:

- `brief.md` — canonical task brief
- `<agent-id>-handoff.md` — specialist result
- `integration.md` — Cortana's synthesis and conflict resolution
- `approval-packet.md` — exact proposal awaiting Joseph's decision
- `outcome.md` — observable result and reusable lesson

Do not store credentials, private keys, seed phrases, exact private financial
positions, or unnecessary personal data.

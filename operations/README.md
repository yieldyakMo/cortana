# Operations Knowledge

This directory contains reusable operating systems recovered from the previous
OpenClaw setup. These are active working references, unlike the historical
records under `archive/legacy-openclaw/`.

## Contents

- `agent-orchestration.md` — routing, delegation, handoff, review, and approval
  protocol for the specialist-agent team
- `memory-review.md` — decision, experiment, and memory review process
- `proactivity-guide.md` — action boundaries, recovery, and opportunity signals
- `templates/` — clean schemas for decisions, experiments, social work, and
  market-risk journals
- `agent-jobs/` — active and completed shared work packets

No template grants authority to publish, moderate, spend, trade, or contact
third parties.

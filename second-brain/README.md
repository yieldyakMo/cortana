# Second Brain

This folder is the durable knowledge base for the workspace.

## Structure

- `journal/` for dated summaries and conversation takeaways
- `concepts/` for evergreen ideas, frameworks, and reference notes
- `documents/` for working docs, plans, and drafts

## Conventions

- Use wiki-links when a note should stay connected to another note
- Keep journal entries short and factual
- Keep concept notes stable and reusable
- Move anything transient back to `memory/`

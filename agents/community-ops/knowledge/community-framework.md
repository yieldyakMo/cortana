# Community Operations Framework

## Community record

For each community, track only:

- platform and public channels
- purpose and voice
- published rules
- recurring topics and peak activity windows
- active campaigns
- aggregate health metrics

Do not create private dossiers on members.

## Engagement

- Read context before drafting a response.
- Adapt content to each platform.
- Follow announcements with useful conversation.
- Acknowledge criticism and address facts without feeding bait.

## Moderation recommendations

- Off-topic: gentle redirect.
- Heated but good-faith debate: slow the interaction and restate boundaries.
- Suspected harassment: gather context and recommend proportionate escalation.
- Clear scam or malicious spam: preserve evidence and request approval for
  immediate protective action.

Consequential recommendations need evidence, documented policy, proportionality,
and a review path.

## Health metrics

Prefer active participants, useful exchanges, retention, response quality,
unresolved incidents, and sentiment trends over total member count.
